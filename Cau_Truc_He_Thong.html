2. Device Service

Quản lý toàn bộ metadata thiết bị.

Table Chức năng
stations Quản lý trạm quan trắc
gateways Quản lý ESP32, Raspberry Pi
sensors Quản lý cảm biến
sensor_types Loại cảm biến
firmware_versions Danh sách firmware
gateway_firmware_history Lịch sử cập nhật firmware

3. Auth Service

Xác thực và phân quyền.

Table Chức năng
users Tài khoản đăng nhập
roles Vai trò
permissions Danh sách quyền
role_permissions Mapping role-permission
refresh_tokens Refresh JWT

4. Sensor Service

Trái tim dữ liệu của hệ thống.

Nhiệm vụ

  • Nhận dữ liệu từ RabbitMQ
  • Lưu InfluxDB
  • Cache dữ liệu mới nhất bằng Redis
  • Realtime WebSocket
  • API truy vấn lịch sử
  • Tính toán Min/Max/Avg
  • Phát hiện Gateway Offline

InfluxDB

Bucket:
sensor_bucket

Measurement:
sensor_data

Tags:
station_id
gateway_id
sensor_id
sensor_type

Fields:
value

Timestamp:
event_time

5. Alert Service

Phát hiện bất thường.

Table Chức năng
alert_rules Luật cảnh báo
alerts Cảnh báo hiện tại
alert_histories Lịch sử cảnh báo

Ví dụ Rule

Temperature > 50°C

Humidity < 20%

CO2 > 1000 ppm

Gateway Offline > 5 phút

6. Notification Service

Table Chức năng
notification_templates Mẫu thông báo
notifications Thông báo
notification_logs Lịch sử gửi

Kênh hỗ trợ

  • Email
  • Telegram
  • SMS
  • Web Notification

7. Redis

Chức năng

  • Cache dữ liệu mới nhất
  • Giảm tải InfluxDB
  • Realtime Dashboard
  • Session Cache

Ví dụ

Key:
latest:TEMP_01

Value:
{
 "value":31.5,
 "time":"2026-06-10T09:30:00"
}
